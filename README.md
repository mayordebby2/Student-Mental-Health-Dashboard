# Student Mental Health Dashboard

## Overview

This project explores the effects of mental health on students' academic performance using Power BI.

---

## Business Problem

Mental health challenges can significantly affect academic performance and student wellbeing.

This analysis identifies trends involving depression, anxiety, panic attacks and academic outcomes.

---

## Tools Used

- Power BI
- Power Query
- DAX
- Data Visualization

## Key Insights

- 35 students reported depression
- 82.86% of depressed students were not receiving treatment
- Panic attacks appeared more frequent among high-performing students
- Year 1 students represented the largest population

---

## Recommendations

- Improve counseling access
- Conduct mental health screening
- Introduce stress management programs
